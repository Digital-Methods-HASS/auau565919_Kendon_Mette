Her bruger jeg pivot_longer funktinen til make the dataset longer by increasing the number of rows and by decreasing the number of columns. Sdan bliver datoerne rykket ind i en lang kolonne og antallet af dde liges. Fordi jeg kun er interesseret i tallene fra Punjab bruger jeg filter-funktionen. Den sidste funktion bruger jeg fordi at R genkender vrdierne i kolonnen som ?? alts bogstaver og jeg vil have den til at genkende det som datoer, sdan at det er nemmere at visualisere.

c19 %>% 
  pivot_longer(all_of(dates), names_to = "date", values_to = "deaths") %>% 
  filter(States == "Punjab") %>% 
  mutate(date = ymd(date)) -> c19

#Analyse/visualisering The following analysis will be based on three visualisations made from the two datasets saved in RStudio as C19 and spanish_flu. The first visualization will show a graph over the amounts of deaths in the initial phase of COVID-19. In order to make this visualization I used the ggplot function, which is a function that allows you to create complex plots from data stored in a data frame (https://datacarpentry.org/r-socialsci/04-ggplot2/index.html). The reason it was so important to make the COVID-19 dataset tidy was also that ggplot works best in tidy data, dvs. a column for every dimension and a row for every observation. (https://datacarpentry.org/r-socialsci/04-ggplot2/index.html). The following command takes its starting point in the c19 dataset and includes the ggplot-function and the geom_point-function, which tells R that I want a plot based on the c19 dataset existing of dot plots. Aes is short for aesthetics and is a function I use to specify how the variables date and deaths should be presented in the graph as x/y positions.

c19 %>% 
  ggplot() +
  geom_point(mapping = aes(x = date, y = deaths))

This is a graph showing the amounts of deaths due to COVID-19 from march to september 2020 in Punjab, India. Grafen viser hvordan antallet af dde i de frste fire mneder steg langsomt indtil midten af juli hvor antallet af dde steg drastisk. I midten af august nr antallet af dde helt op over 800, og viser hvordan COVID-19 i perioden har taget ordentligt fat i Punjab her.
